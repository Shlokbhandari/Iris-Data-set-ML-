# Iris Dataset – Machine Learning

This repository contains my machine learning practical work using the Iris dataset.

## Contents
- `iris_training.ipynb` – Jupyter notebook for training a model on Iris data  
- `iris.xlsx`, `iris-train.xlsx`, `iris-test.xlsx` – Dataset files  
- `.ipynb_checkpoints/` – Auto-saved notebook checkpoints

## Tools Used
- Python
- Jupyter Notebook
- NumPy, Pandas, Scikit-learn

## Purpose
This project is for learning and practicing basic machine learning concepts.
